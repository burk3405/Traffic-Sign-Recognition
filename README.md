# Traffic-Sign-Recognition
Computer Vision project on detecting and recognizing Traffic Signs

1st Step: Review literature of common techniques used for sign detection and recognition, and challenges of real-world imagery. 
2nd Step: Create a system design and methodology. This includes image descriptions, pre-processing, a detection phase, a recognition phase, and implementation.
3rd Step: Experimental results, including dataset descriptions and evaluation metrics.
4th Step: Visualize findings.
5th Step: Discussion and conclusion.
